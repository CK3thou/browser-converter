## üîß Troubleshooting

**Extension doesn't appear?**
- Make sure you clicked "Load unpacked"
- Make sure you selected the correct folder
- Refresh the page (Ctrl+R)

**Popup doesn't show?**
- Try highlighting different text
- Refresh the webpage (Ctrl+R)
- Check console for errors (F12)

**Settings won't open?**
- Click the extension icon again
- Make sure it's enabled in chrome://extensions/

**Conversions look wrong?**
- Check that you highlighted valid text (e.g., "USD 100")
- Make sure you have internet for live rates
- Check the API is working in console
